# clone-cards

Proyecto de práctica: clon de una interfaz de tarjetas (cards) usando React + Vite. Ideal como plantilla para ejercicios y demos: componentes funcionales, CSS simple y flujo de desarrollo con Vite.

## Descripción
Pequeña SPA que muestra tarjetas (cards) con componentes reutilizables y estilos CSS. Pensado para practicar React, estructura de componentes y despliegue básico.

## Tecnologías
- React 19
- Vite
- @vitejs/plugin-react-swc
- ESLint

## Requisitos
- Node.js 16+ (recomendado)
- npm (o `yarn` / `pnpm`)

## Instalación rápida
```bash
git clone <tu-repo-url>
cd clone-cards
npm install
